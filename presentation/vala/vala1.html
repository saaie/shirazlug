What is Vala?

  • Compiler for the GObject type system
  • Vala is a programming language like C#
  • Vala is an Object-Oriented programming

Why Another Language?

  • Native coding
  • Simple coding with simmilar performance coding in C
  • Compile-time gorbage collector
  • 100% compatible with C
  • Access pre-existing libraries without runtime wrapper
  • Built-in GI support
  • Borned to be async
